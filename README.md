Calculadora Fiscal DIFAL & FCP
Projeto desenvolvido para simulação de Diferencial de Alíquota (DIFAL) e Fundo de Combate à Pobreza (FCP).

Funcionalidades:

Identificação automática de Base Dupla (Por Dentro) ou Base Simples (Por Fora).

Alíquotas interestaduais automáticas baseadas na UF Origem/Destino.

Memória de cálculo detalhada passo a passo conforme Nota Técnica 2020.005.

Máscara de moeda em tempo real.
